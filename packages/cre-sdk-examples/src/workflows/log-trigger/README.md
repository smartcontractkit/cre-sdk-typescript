# Log Trigger Workflow

This workflow is a simple example of how to use the log trigger capability to listen for events on a given contract.

## Usage

```bash
cre workflow simulate ./src/workflows/log-trigger
```

## Configuration

You can use this value when prompted for by log trigger simulator:

```
Transaction Hash: 0x9394cc015736e536da215c31e4f59486a8d85f4cfc3641e309bf00c34b2bf410
Log Event Index: 0
```
